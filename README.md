# Cinémathèque — The Premium Cinema Experience 🎬

**Cinémathèque** is a high-fidelity, community-driven movie discovery platform. It provides an immersive cinematic experience by blending real-time data from the TMDB/OMDB APIs with a luxury "Obsidian and Gold" user interface. 

Built for the **TEXPERIA 2026** competition, this project focuses on high-end UI/UX, responsive performance, and interactive community features.

---

## 📽️ Problem Statement
Traditional movie platforms often feel cluttered, slow, and purely transactional. Most lack a sense of "theatrical atmosphere" and fail to bridge the gap between movie discovery and community discussion.

**Cinémathèque solves this by:**
* **Thematic Immersion:** Using a dark-mode luxury aesthetic to simulate the theater environment.
* **Real-time Integration:** Dynamically fetching movie data, trailers, and cast information.
* **Social Connectivity:** Providing a dedicated space for cinephiles to discuss theories and reviews.
* **Seamless Navigation:** Implementing a Single Page Application (SPA) architecture for zero-latency transitions.

---

## ✨ Key Features

### 💎 Premium UI/UX
* **Cinematic Landing Curtain:** A dramatic 2-second theater-style curtain lift upon page load.
* **Ken Burns Effect:** Slow, deliberate zooming on hero backgrounds for a documentary-style feel.
* **Staggered Entrance Animations:** Elements glide into place sequentially using custom CSS cubic-beziers.
* **Red Glow Interactions:** Interactive "Scene Stills" and cast profiles that highlight and glow in crimson on hover.

### 🔍 Discovery & Personalization
* **Global Search:** Real-time filtering across thousands of films via the TMDB API.
* **Categorization:** Intelligent sorting by Top Rated, Trending, and Genre.
* **Watchlist:** A persistent "My List" feature to save cinematic experiences for later.
* **Infinite Scroll:** Optimized "discover" grid that loads more content as you reach the end of the page.

### 👥 Community Hub
* **Discussion Boards:** A platform to share theories, reviews, and film scores.
* **Top Reviewers Leaderboard:** Gamifying the critic experience for community members.
* **Trending Topics:** Hashtag-driven navigation for what's currently hot in cinema.

---

## 🛠️ Technical Stack
* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+).
* **APIs:** TMDB (The Movie Database) & OMDB (Open Movie Database).
* **Animation Libraries:** AOS (Animate on Scroll).
* **Icons/Fonts:** FontAwesome 6.5, Playfair Display, DM Sans, Bebas Neue.

## 📱 Responsiveness
The site is fully responsive across three breakpoints:
* **Desktop:** Full 6-column grid with hover interactions.
* **Tablet:** Condensed 3-column grid with touch-swipe enabled carousels.
* **Mobile:** 2-column simplified layout with optimized typography for readability.
